A simple Qt5 app showing your system uptime


Install required libs and tools:
```
sudo apt install qtbase5-dev qtcreator qtbase5-dev qt5-qmake libqt5widgets5 build-essential
```

run qmake:
```
qmake
```

then run make:
```
make
```

run the program:
```
./uptime-show
```
